# Setup and running

```bash
# In folder app do
$ yarn
$ yarn start
```

```bash
# In root folder do
$ yarn
$ npm i -g cypress
# In folder cypress do
$ cypress open .
```

# Available commands
```
- reload
- visit <url>
- click
- force click
- <element> contains <content>
- input <what> into <element>
- does <element> exists
- run <macro>
- chain <chain name> on <element>
```
